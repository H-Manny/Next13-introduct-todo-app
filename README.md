## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.<br>

run JSON SERVER:
```
npm run json-server
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
